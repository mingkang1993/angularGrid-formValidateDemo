gridApi

gridOptions表格参数

method ---String ---- 请求类型,'GET','POST'

url ---String ---- 请求的url

columnDefs ---JSON ---- 表格头部参数-映射数据,详情看下面

params ---Object ---- 传到后端的参数,默认参数pageNo:1pageSize:10

selectAll ---Boolean ---- 是否显示全选,默认false

showPage ---Boolean ---- 是否显示分页,默认true

data ---JSON ---- 每页数据

selectData ---JSON ---- 选中的数据

paginationCurrentPage ---number ---- 当前的page

paginationPageSize ---number ---- psgeSize

columnDefs表格头参数-映射数据

displayName ---String ---- 表格的title

width ---String ---- 表格宽度

field ---String ---- 数据映射,对应后台返回对应的参数key,优先级最低

cellTemplate ---String ---- 自定义模版拼接,有row字段,优先级中等

filter ---FN(row) ---- 前两者都无法满足的时候,可以选用filter,最后需要return出来模版,优先级最高

gridCallback

gridOptions.search ---FN(Callback,searchOpt) ---- 表格搜索的时候

gridOptions.refresh ---FN(Callback) ---- 刷新当前页面,page不变

gridOptions.loadSuccessCallback ---FN(data) ---- 每页数据请求完callback

gridOptions.pageSuccessCallback ---FN(data) ---- 每页点击请求完callback

后端接受参数

pageNo ---number ---- 页码

pageSize ---number ---- 每页条数
